Vulos Office

Documents, sheets, slides & e-signatures — in one self-hostable workspace.

Vulos Office is an open, self-hostable document suite: real-time collaborative docs, spreadsheets, and presentations with version history, comments, suggestions, and a built-in signing flow. One Go binary, every asset embedded — your documents, on your infrastructure.

Sign in View on GitHub

Everything in one binary

Self-hostable, open source, and built to run anywhere.

Docs, sheets & slides

Write rich documents, crunch spreadsheets, and build presentations — with PDF, DOCX, XLSX, and PPTX export.

Real-time collaboration

Live co-editing with version history, named snapshots, an activity feed, threaded comments, and track-changes suggestions.

E-signatures

Place fields, send signing links, orchestrate multiple signers, and download sealed PDFs with a verifiable audit manifest.

Single binary

The SPA and every asset are embedded via embed.FS; durable state in SQLite by default, with an optional Postgres backend for shared deploys.

Security-first

JWT sessions over an HttpOnly cookie, per-file access control, invite-only registration, and rate-limited write endpoints.

Apps & bots

An open MCP server and apps platform let any LLM or agent operate your documents over the same access-controlled seam.

Part of Vulos

Vulos is an open, self-hostable web OS and app suite. Vulos Office runs perfectly standalone — and can be combined under one login alongside Vulos Mail, Talk, Meet, and more by Vulos Workspace.
